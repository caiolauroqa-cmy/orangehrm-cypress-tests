
# 🧪 Testes Automatizados - OrangeHRM

Projeto de automação de testes E2E utilizando Cypress no sistema OrangeHRM.

## 📌 Objetivo
Validar funcionalidades críticas do sistema, garantindo qualidade e confiabilidade, através de testes automatizados.

---

## 🚀 Funcionalidades testadas

- Login de usuário (cenários positivos e negativos)
- Busca de funcionários por nome
- Busca de funcionários por ID
- Filtros por status e cargo
- Reset de filtros
- Cadastro de funcionário

---

## 🛠️ Tecnologias utilizadas

- Cypress
- JavaScript
- BDD (Cucumber)
- Page Objects
- Git
